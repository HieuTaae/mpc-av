# Model Predictive Control Project

---

## Dependencies
* cmake >= 3.5
* make >= 4.1
* gcc/g++ >= 5.4
* [uWebSockets](https://github.com/uWebSockets/uWebSockets)
  * Run `install-mac.sh`
* Ipopt and CppAD
* CppAD

## Build Instructions
1. Clone this repo.
2. `mkdir build && cd build`
3. `cmake .. && make`
4. `./mpc`.
5. Open the simulator
